## Prerequisites

  * [Helm] ^3.8.0

[Helm]: https://helm.sh/

## Charts

Add the raccoon repository to Helm:

```shell
helm repo add raccoon https://charts.raccoon.team
helm repo update
helm search repo raccoon
````
